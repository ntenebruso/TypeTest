# Current Features

-   15, 30, and 60 second WPM tests
-   Deep analysis:
    -   WPM calculation
    -   Accuracy calculation
    -   Correct/incorrect words ratio
    -   Correct/incorrect characters ratio
    -   WPM and error count interactive chart
    -   User account system and leaderboard
